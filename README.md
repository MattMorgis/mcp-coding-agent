# Simple CLI Agent

A simple CLI-based agent built with the mcp-agent framework.

## Setup

1. Install dependencies:

   ```
   uv sync
   ```

2. Set your Anthropic API key as an environment variable:
   ```
   cp mcp_agent.secrets-example.yaml mcp_agent.secrets.yaml
   ```

## Usage

Run the agent

```
uv run main.py
```
